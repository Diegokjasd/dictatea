    Tablero de comunicación interactivo para menores con TEA

    Desarrollamos una aplicación intuitiva para ayudar a los niños con trastorno del espectro autista (TEA) con dificultad de comunicación el poder tener una forma de comunicarse con su familia y amigos para así siempre poder mantenernos comunicados

    ¿Como funciona una tabla de comunicacion?

    Las tablas de comunicacion tienen como funcion principal la comunicacion de un individuo con otro. El tablero permite facilitar la comunicacion de una persona con dificultades de habla, dificultades de expresarse, entre otros.

    Para poder comunicarse, unicamente debe presionar los dibujos que representan acciones, sentimientos, objetos, lugares, etc. La aplicación dira lo que presiono con el tipo de voz elegido masculino o femenino

    A continuacion veras una tabla de comunicacion fisica:

    Al igual que una tabla de comunicacion fisica, con DictaTEA usted debera señalar y presionar lo que desea comunicar. DictaTEA comunicara lo que desee transmitir con voz.
